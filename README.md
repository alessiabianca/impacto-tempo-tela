# impacto-tempo-tela
Ferramenta web para cálculo quantitativo de tempo de tela e triagem de saúde digital.
# 📱 Sistema Interativo de Triagem de Saúde Digital

## 📌 Sobre o Projeto
Ferramenta computacional desenvolvida para análise quantitativa de uso de dispositivos móveis e seu impacto no corpo humano. O sistema foi projetado e aplicado como recurso tecnológico na **Amostra Pedagógica Escolar (2026)**, focado na conscientização sobre a Síndrome Metabólica, higiene do sono e ergonomia.

## ⚙️ Funcionalidades
- **Cálculo de Projeção em Tempo Real:** Converte horas diárias de uso de telas em métricas de impacto de longo prazo (dias por ano e anos de vida perdidos).
- **Interface de Diagnóstico:** Sistema de slides integrados (`Dark Mode` de alto contraste) para exposição de conceitos científicos.
- **Triagem:** Suporte visual para o preenchimento de "Planos de Desintoxicação Digital" com base em faixas de tempo pré-estabelecidas.

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3:** Estruturação semântica e estilização (Layouts responsivos, Grid/Flexbox).
- **JavaScript (Vanilla):** Lógica matemática para processamento instantâneo de variáveis temporais no *client-side*.

## 🎓 Relevância Acadêmica e Extensão
Este projeto atua na interseção entre a **Computação Educacional e a Saúde Preventiva**, demonstrando a aplicação de lógica de programação para resolver problemas práticos de conscientização comunitária. O uso da gamificação e da matemática aplicada facilita a compreensão de dados biométricos abstratos pelo público geral.
